# dfjinxin-fast-vue

## 代码下载后在工程当前路径运行，下载所有依赖包
```
yarn install
```

### 运行工程
```
yarn run serve
```

### 工程打包
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
